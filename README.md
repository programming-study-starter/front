## What to do?

- 게시판 서비스로 시작하여 결재문서 상신까지 할 수 있도록한다
- 회원이 사용할 수 있는 게시판
- 회원의 등급 별로 할 수 있는 행위를 제한할 수 있도록

## How to run

```bash
npm install

npm run dev # localhost:3000

npm run build

npm run start
```
