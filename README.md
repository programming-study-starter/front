# Front Project
- Next.js 를 활용한 Front end project

## Subject
- 아직 무엇을 할 지 결정하지 못했음

## Skill
- Next.js
- axios

## How to run
```bash
npm install

npm run dev # localhost:3000

npm run build

npm run start
```
